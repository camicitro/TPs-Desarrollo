# TP Typescript
En el presente TP, realicé la consumisión de una API pública (FakeStoreApi) mediante fetch API.
De la API obtenemos 20 productos que luego serán mostrados en una tabla. Pero previo a eso, hay un spinner de carga que se muestra hasta obtener los 20 productos y crear la tabla, luego de que se consiguió, el spinner desaparece.
El código fue realizado siguiendo los videos sobre el TP.
En el archivo index.html, se utilizó bootstrap, y dentro del mismo se encuentra el código CSS, no hice un archivo CSS aparte.
En el archivo index.ts realicé la llamada al endpoint de productos, y allí mismo fui guardandolos en la tabla. 

### Aclaraciones:
- La carpeta de node_modules la comprimí para subirla ya que no me permitía subirla por su contenido
- Los archivos de types, no fueron usados para la realización del TP, solo fueron para pruebas previas
